历史上的今天 09-26

dapeng

2017-09-26 12:39 GMT

原文链接: steemit, cnsteem, chainbb, busy, markdown

不久之前，搞过一个”大鹏观察-历史上的今天”，俗称“挖坟帖”“怀旧帖”“八卦帖”“忆苦思甜帖”，口号是“带你揭开历史的画卷，钩起尘封的往事”。你会看到，在去年的今天，@laoyao 说：“哈哈，终于断篇了，说明steemit进入谷底。”而 @oflyhigh 说：“非也…蹲的越深，跳的越高…steemit正在积蓄力量。”

@tumutanzi 曾经留言问：这个帖子有什么意义？我回答说：暂时不好说，但是说不定做着做着就有了意义。

后来，这个栏目因为种种原因停办了。昨天，这事儿又被 @justyy 提出来了。于是，我汇总了一下大家的声音：

  • @deanliu说：好懷舊的感覺啊~~~

  • @tvb 说：我觉得挖出“骨灰贴”还是有意义的。我也想知道那些我关心的朋友和前辈们，曾经写过什么文章，曾经有过什么想法。对于还在起步阶段的我，也许是一种启发；对于认识这些朋友们，也可以更丰富、更深入。所以我觉得这个有继续下去的必要的。也许挖出来的贴并都是我感兴趣的。但是，总有一些精彩，是我错过的！

  • @luxurywalker 说：感谢您汇集这些数据！实在是很有用！

  • @justyy 说： @dapeng 之前搞了一个 “历史上的今天” 挖坟贴 ，也就是通过 SQL 查询 在以前在同一天发表的帖子，听起来很玄乎……

  • @daijia 说：哪怕只能帮助到一个人也有意义了。

听起来还是有点意义的。所以，“历史上的今天”重新开张！

今天是 2017 年九月 26 日，星期二。历史上的九月 26 日，steem 的 “cn” 板块发生过什么事？
